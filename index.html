🎪
01 — BRANCHE
Van de branche, voor de branche
Ontwikkeld vanuit echte festivalproductie. Elk scherm, elke knop en elke workflow is afgestemd op hoe productieteams daadwerkelijk werken.
⚡
02 — SNELHEID
Binnen vijf minuten live
Project aanmaken, lineup invullen, links delen. Je hele team staat op hetzelfde schema zonder installaties, trainingen of langdurige onboarding.

Live of handmatige start

Alles wat je nodig hebt
op de festivaldag

Gebouwd voor productieteams die op de dag zelf geen tijd hebben voor gedoe.

🔴
Live & handmatige show start
Automatisch op geplande starttijd of handmatig getriggerd door jou — per show instelbaar. Volledige controle, geen verrassingen.
📡
Realtime sync
Elke wijziging is direct zichtbaar op alle apparaten van je team. Geen refreshes, geen verwarring — altijd het laatste schema.
⏱
Countdown timers
Automatische aftelling per act. FOH, backstage en stage view tonen elk hun eigen relevante timer.
🎭
Backstage view
Deelbare link voor backstage crew. Geen app nodig, werkt op elke telefoon via de browser — direct open, direct bruikbaar.
📺
FOH overzicht
Volledig productiescherm voor aan de mix. Notities, countdown per show, live status van elke act op je stage.
📅
Meerdaags festival
Vrijdag, zaterdag, zondag — elk met eigen lineup, eigen stages. Timer loopt alleen op de dag dat het ook echt zover is.
📱
Installeerbaar als app
Voeg toe aan je thuisscherm via Safari of Chrome. Werkt als native app zonder App Store of Play Store.
📥
JSON import
Heb je al een lineup? Plak die als JSON en hij maakt stages, dagen en shows in één klap automatisch aan.
👥
Teambeheer
Voeg gebruikers toe aan je bedrijf. Iedereen krijgt toegang tot de juiste projecten, jij behoudt overzicht als admin.

live op de dag zelf

Van aanmaken tot je hele crew op hetzelfde schema — binnen minuten.

01
Maak een project aan

02
